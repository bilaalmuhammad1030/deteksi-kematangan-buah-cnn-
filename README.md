# Fruit-Ripeness-Detection-using-CNN-Xception-
Proyek ini mengembangkan sistem deteksi tingkat kematangan buah berbasis citra digital menggunakan CNN dengan arsitektur Xception. Dataset diklasifikasikan ke dalam tiga kelas, yaitu unripe, ripe, dan overripe. Metode transfer learning, preprocessing, dan augmentasi citra diterapkan untuk meningkatkan performa dan akurasi model.
